🍞 Toastlet Notify

Modern and responsive floating notification JavaScript library

Zero Dependencies ~8KB Minified Mobile Ready ES5+ Compatible

🚀 Installation & Usage

Get started with Toastlet Notify in seconds. Just include the CDN link and you're ready to go!

📦 Via CDN (Recommended)

<script src="https://cdn.jsdelivr.net/gh/pedrohrigolin/Toastlet-Notify-JS@main/toastletNotify.min.js"></script>

⚡ Basic Usage

// Simple notification toastletNotify.notify('success', 'Operation completed successfully!'); // With options toastletNotify.notify('warning', 'Please check your input', { delay: 10000, sticky: true });

🎨 Custom CSS Example

/* Dark theme example */ .toastlet.dark-theme { background: linear-gradient(135deg, #2c3e50 0%, #34495e 100%) !important; color: #ecf0f1 !important; border-left: 4px solid #3498db !important; } /* Usage */ toastletNotify.notify('success', 'Custom styled notification!', { class: 'dark-theme' });
